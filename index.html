Project Hoger-Lager:

Bij dit project begon ik net met het leren van html, css en javascript dus ik wist nog niet zoveel van alles. Het heeft me erg veel geholpen om css te begrijpen omdat we de kans kregen om veel kleuren toe te voegen en borders en fonts uit te proberen.

Het doel van het project was om het spel hoger-lager na te maken. Er waren bepaalde eisen waar we ons aan moesten houden en moesten volgen om het spel succesvol na te maken.

Project Zeeslag:

Dit was ons tweede project en het was deel van build-a-game wat ik samen met Damian heb gemaakt. Ik heb bij dit project vooral javascript geleerd omdat je veel dingen moest en kon doen met javascript en het lukte allemaal best wel goed. Ik heb uiteindelijk door het oefenen met javascript voor het project ook een erg mooi cijfer gekregen voor de toets.

Het doel van het project was om aansluitend bij build a game een game te maken, ik heb gekozen voor zeeslag dus ik moest het traditionele spel zeeslag online namaken.

Project Escaperoom:

Dit project was deel van onze eerste themaweek waar ik met Adrian en twee andere klasgenoten een esacperoom heb gemaakt. Bij dit project heb ik veel van css en javascript geleerd en geleerd hoe je moet samenwerken met mensen die je niet goed kent. Css heb ik geleerd en mee geoefend omdat ik alle css van de hele game heb gemaakt en toegepast en javascript omdat ik een spel had gemaakt die helemaal werkte op javascript.

Het doel van het project was om samen met 3 andere een escaperoom te maken. Aangezien we met 4 mensen waren maakte we 4 puzzles die je moest ontcijferen en halen om de code voor de deur te krijgen. Uiteindelijk hebbn we ook een uitleg video erbij gemaakt en ingeleverd.

Project Build-A-Game:

Dit project hoorde ds bij zeeslag en de game van Damian wreck-a-mole. Ik heb veel geleerd van html en css voor het hoofd menu scherm en het door scherm omat ik die heel mooi heb versierd met css.

Het doel van het project was om samen in een groep een paar games te maken die in het echt ook bestaan. Het ging erg goed en ik heb ook een mooi cijfer gekregen.
